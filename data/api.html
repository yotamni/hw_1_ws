VOD Web Service New Serieses API

Overview

This page shows sample requests to the VOD new serieses Data API. You use the VOD Data API to retrieve and manipulate VOD resources. Each sample on this page show the method, the URL and the parameters you need to use. The response to each request is the JSON representation of a VOD resource. The response includes the serieses details like the example below.

EXAMPLE
{
serieses: [
{
id: 2,
name: "Fargo",
genre: "Thriller",
season: 3,
episode_number: 10,
rating: 9
}
]
}

Retrive All Serieses New

This request retrive all the new serieses in the VOD.
In case that no series is available the service retive {serieses: { err: "series not found" }}.

GET https://new-serieses.herokuapp.com/getAllNewSerieses

Retrive Serieses By Genre New

This request retrive the serieses belongs to specific genre. This request use the POST method and pass the parameter genre in the body request. The genre options are [ Crime, Thriller, Comedy, Mystery, Horror, Drama, Reality, History ].
In case that no series in the requested genre found the service retive {serieses: { err: "series not found" }}.

POST https://new-serieses.herokuapp.com/getSeriesByGenre { parameter key = genre }

Retrive Serieses By Rating New

This request retrive the serieses that their raiting level is betwin the two given parameters. The :low parameter represents the lowest reiting level requested and the :high parameter represents the highest rating level requested. The service retrive all serieses with raiting value betwin this limits. The rating values range is 1-9.
In case that :low greater then :high the service retrive {serieses: { err: "parameters not in order" }}.

GET https://new-serieses.herokuapp.com/getSeriesByRating/:low/:high
